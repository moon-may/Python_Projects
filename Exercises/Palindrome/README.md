## Поиск палиндрома

### Условие
На вход подается строка. Необходимо определить, является ли она палиндромом (читается одинаково как слева направо, так и наоборот). Регистр букв, пробелы и знаки препинания не имеют значения. 

### Пример входных данных
А роза упала на лапу Азора  
Привет, мир!

### Пример вывода
True  
False
