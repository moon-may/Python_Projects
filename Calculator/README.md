## Калькулятор с GUI
Рабочий калькулятор с графическим интерфейсом. Реализован базовый функционал: арифметические действия, возведение в степень, квадратный корень, работа с памятью.  
Инструментарий: Python 3, Tkinter (GUI).  

### Планы по доработке
  - **Обработка ошибок**: отлавливается только ZeroDivisionError
  - **Расширение функционала**: тригонометрия, логарифмы
